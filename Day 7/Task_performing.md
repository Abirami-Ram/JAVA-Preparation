## 🔍 Relational operator 

- It is a binary operator 
- It is used to compare two values 
- After comparing it is going to return as a Boolean value 

### The relational operators are - 6 symbols

```
>   Greater than
<   Less than

<=  Less than or equal to 
==  Equal to

```

## ❓ Conditional Operator

- It is a ternary operator
- It is used to make decision based on a condition

### What is the condition?

Condition means Boolean value or anything that is going to return a Boolean value.

### Syntax:
```
Operand1 ? Operand2 : Operand3
```

- Operand1 -> condition
- If condition is true, operand2 will get executed.
- If condition is false operand3 will get executed.

👉 We use 3 operands, so it is ternary operator.

---

## 🔗 Logical Operator

- These are the operator which is used to combine two or more conditions and return a Boolean value 
- It helps to check multiple conditions at the same time.

### 3 types of logical operators:

```
&&  - AND  – if both conditions are true, it will be true otherwise false  - Binary operator
||  - OR   – if one condition is true it will be true otherwise false – Binary operator 
!   - NOT  – opposite operator - Unary operator 
```

## 🔼 Increment and Decrement 

- Increment ++
- Decrement --

👉 They are unary operators. we use the symbol with the help of one operand

- Increment – It is going to increase the value of a variable by 1.
- Decrement – It is going to decrease the value of a variable by 1.

### Increment is divided into 2 types 
- Post Increment 
- Pre Increment

### Decrement is divided into 2 types 
- Post Decrement 
- Pre Decrement

---

### 🔹 Post Increment:

First use the existing variable and then increment the original value  

```
Variable++
```

Eg : 
```
a++
```

---

### 🔹 Pre Increment:

First increment the original variable and then use the updated value  

```
++Variable
```

Eg :
```
++a
```

---

### 🔹 Post Decrement:

First use the existing variable and then decrement the original value  

```
Variable--
```

Eg :
```
a--
```

---

### 🔹 Pre Decrement:

First decrement the original variable and then use the updated value  

```
--Variable
```

Eg :
```
--a
```

It is a combination of Arithmetic and Assignment operator.

```

/=
%=
```

### Eg :
```java
int a = 10;
a += 20;   or a = a + 20;
```

## 📌 Assignment operator 

Assignment operator is going to assign the value to the variable

