# 📘 Operator

---

## 🧩 Operator:
Operator is a specific symbol that tells the computer to perform some specific action.

### 💻 Eg:
```
8+9
```
👉 Here + is an operator 

---

## 🔢 Operand:
Operands are value or variable in which an operator works.

### 💻 Eg:
```
8+9
```
👉 Here 8 and 9 are operand

---

## 🔗 Expression:
Combination of operator and operand is known as expression.

---

## ⚙️ Operation:
When an operator is used with operands, the task it performs is called an operation

---

## ✨ Characteristics of operator:

### 🔹 Operators return a result 
👉 Eg :
```
2+5
```
👉 after doing the task it going to return result as 7 

👉 Result can be numeric ,Boolean  another type depending on the operator used 

---

### 🔹 Every operator have a precedence(priority)

👉 Eg :
```java
int x= 2+3*4 
```

👉 BODMAS RULE 

```
x=2+12=14
```

---

### 🔹 operators have associativity 

👉 Eg :
```
10+3-4
```

👉 + and – have same precedence then you have to follow left to right associativity
