# 📘 Number of Operands

---

## 🔢 Number of operands :

---

## 1️⃣ Unary operator 

👉 The operators that work on only one operand  

### 💻 Eg :
```java
a++
```

---

## 2️⃣ Binary operator 

👉 The operator that works on two operands  

### 💻 Eg :
```java
a + b
```

---

## 3️⃣ Ternary operator 

👉 The operator that works on three operands  

### 💻 Eg :
```java
a > b ? True : False;
```
