# 📘 Tokens

---

## 🧩 Definition
👉 Smallest individual unit of a programming language.

---

## 💻 Example
```java
class Demo 
{ 
public static void main(String[] args) 
{ 
} 
}
```

👉 Here each and everything written is a token.

---

## 🔍 Different types of tokens

- 🔑 Keywords 
- 🏷️ Identifiers 
- 🔢 Literals 

---

## 🔑 Keywords

👉 In java they are few words that have predefined meaning and easily understandable by the compiler is known as keyword.

- 📌 There are 50 keywords in java.
- ✔️ Example: `class`, `public`, `static`, `void`
- 🔤 They are written in lower case.

👉 Important:
- ❗ `main` is not a keyword it is a name given to the method.

---

## 🏷️ Identifiers

👉 It is the name give by the user to the programming language.

- class 
- variables 
- method 
- interface 
- Constructor

### 💡 Example
```java
int a;
```
👉 here `a` is identifier

---

## 📏 Rules

- ❌ Keywords cannot be an identifier.
- ❌ We cannot have numbers in the initial position.

```java
class 7Demo   // ❌ Invalid
```

- ❌ We cannot have any special characters except `$` and `_`

---

## 🔢 Literals

👉 It is the data / value given by the programmer in a program is known as Literals

---

## 📊 Types of Literals

### 1️⃣ Number Literals

- Integer numbers : `1, 2, 4, 7, 8, 9`
- Decimal numbers : `2.3, 4.5, 7.8`

---

### 2️⃣ Character Literals

- 📌 whenever we are writing character literals it should be enclosed within single quotes.
- 📌 Its length should always be 1.

✔️ Example:
```java
'a', 'e'
```

---

### 3️⃣ String Literals

- 📌 The group of characters is called as string literal.
- 📌 It should be enclosed within double quotes.
- 📌 The length of string literal is infinite.

✔️ Example:
```java
"Hello"
```

---

### 4️⃣ Boolean Literals

- 📌 A Boolean literal in Java represents only two values:
  - `true`
  - `false`
