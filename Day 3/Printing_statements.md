# 🖨️ PRINTING STATEMENTS

---

## 📌 We have two printing statements

---

### 1️⃣ System.out.print(data);

- It have only one task that is to print the passed data.  

- Passing of data is mandatory.  

---

### 📌 Example

```java
class Book {

public static void main(String[] args)

{

    System.out.print("Hello");

}

}
```

---

### 📤 Output

```
Hello
```

---

### 2️⃣ System.out.println(data);

- It have two tasks.  

  - ➤ To print the passed data.  
  - ➤ To move the cursor or control to next line or new line.  

- Passing of data is optional.  

---

### 📌 Example

```java
class Book {

public static void main(String[] args)

{

    System.out.print("welcome");

    System.out.println("Hello");

    System.out.println();

    System.out.println("Bye");

}

}
```

---

### 📤 Output

```
welcomeHello

Bye
```
