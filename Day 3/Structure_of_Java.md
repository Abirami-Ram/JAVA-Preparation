
### 📘 Structure of Java

The file to code the program(set of instruction) is called **source file**.

The extension of source file should be **.java** extension. Then only system understand this is a java file.

**Eg :** `Book.java`

Inside the source file the class will be presented.

---

## 🧩 Syntax for class file

```java
class ClassName
{
    // open of block

    #statement    Class Block

}   // End of block
```

---

## 💡 Example

```java
class Book
{
    //statement
}
```

---

## ✅ Convention

Not compulsory but technical standard that we need to follow.

- The class name and name of source file should be same.
- The first letter of every word of a class name should be capital letter.
