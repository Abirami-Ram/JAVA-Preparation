# 📘 Datatype

\---

## 📌 Definition

Datatype :

&#x20;   It tells us  what type of data we need to store.



\---

## 🔀 Types of Datatype

Datatype divided into 👇

* Primitive datatype
* Non-Primitive datatype

\---

## 🔢 Primitive Datatype

It helps us to store primitive data.

### 📦 Primitive Data :

Data that stores in single containers

To store that data in single container is known as primitive datatype

👉 Types: - Number - Character - Boolean

✔️ They can be stored in a single container

\---

## 🔥 8 Primitive Datatypes

All these 8 primitive datatypes are keywords.

``` java
byte,short,int,long,float,double,char,boolean.
```

\---

## 🧩 Non Primitive Datatype

It helps to store multiple values 📦📦

Eg :

* String
* Array
* all the class that we are creating

Eg :

``` java
class Demo 
{
    Demo d;
}
```

📌 We can create variable with the help of class name.

➡️ Inside this variable we can store: - null

* address
