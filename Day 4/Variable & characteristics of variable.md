# 📘 Variable

------------------------------------------------------------------------

## 📌 Definition

Variable :

    It is a named block of memory which is used to store or fetch the data 

------------------------------------------------------------------------

## ✨ Characteristics of variable

1️⃣ We can store or fetch the data with the help of name

2️⃣ We can store only one value inside a variable

3️⃣ The data stored in a variable is not permanent it is temporary

Eg : int age = 25;

is temporary data

The data (value) does not stay forever

It exists only while the program is running

Once the program stops → data is gone ❌

4️⃣ Every variable has a lifespan

            his talks about the variable itself

Simple meaning:

    The variable (container) exists only for a certain time 

    It is created → used → destroyed 🔄

Eg :

``` java
void myFunction() {
int age = 25; // variable created
} // variable destroyed here
```

------------------------------------------------------------------------

## 🧩 Syntax to create a variable

    Datatype variablename ;

    Int a ;

    String b;

📌 Datatype : It tells what type of data we are going to store

------------------------------------------------------------------------

## 🔢 Syntax to create multiple variables

    int a;

                int b;    int a,b,c,d;

    int c;

    int d;

------------------------------------------------------------------------

## 📖 Declaration

    Datatype Variablename ;

    int a;

------------------------------------------------------------------------

## ⚙️ Initialization

    Variablename = value/expression;

        ‘=’ assignment operator 

------------------------------------------------------------------------

## 🔗 Declaration and initialization

    Datatype variablename = value/expression

                        int a = 10;    a    10

📦 container named as 'a' and its stores the value 10

➡️ assignment operator going to take 10 and store it inside a variable
'a'

``` java
Class Variable {
```
